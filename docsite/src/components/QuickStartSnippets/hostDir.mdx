For `STACKS_FROM=dir` this should be the immediate parent directory containing all your project folders.

<details>

<summary>Example</summary>

```
.
└── /home/myUser/homelab/
    ├── immich/
    │   └── compose.yaml
    ├── plex/
    │   ├── docker-compose.yaml
    │   └── .git
    └── uptime-kuma/
        └── compose.yaml
```

Then **Host Directory** should be `/home/myUser/homelab`

</details>
